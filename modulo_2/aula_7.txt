Nessa aula, aprendemos mais duas condições: o "se não" e o "se não, se".

Na estrutura se (condição) então (recompensa), podemos adicionar um se não (consequência). Além disso, é possível utilizar se não, se (segunda condição) então (segunda recompensa).

Exemplo:

Se (eu arrumar meu quarto)
Então (posso jogar bola)
Se não (vou apanhar)

Se (eu arrumar meu quarto)
Então (posso jogar bola)
Se não, se (lavar a louça)
Então (posso jogar videogame)
Se não (vou apanhar)

Também é importante identar o código da maneira correta para mantê-lo organizado. Mesmo que o Lua entenda um código desorganizado, é sempre bom formatá-lo corretamente. Caso tenha dúvidas, existem dois sites que podem ajudar:

Beautifier (organiza o código corretamente):
