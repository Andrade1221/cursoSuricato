Nessa aula aprendemos mais duas condições que seria o se não e o se não, se. onde temos no se (condição) então (recompensa) ai vem o se não (consequencia) e também tem como por o se não, se (segunda recompensa). Exemplo:

Se ( Eu arrumar meu quarto )
Então ( Posso jogar bola )
Se não ( Vou apanhar )

Se ( Eu arrumar meu quarto )
Então ( Posso jogar bola )
Se não, se ( Lavar a louça )
Então ( Posso jogar no video game )
Se não ( Vou apanhar )
